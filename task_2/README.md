# Задача 2. Pizza Toppings

## Условие

Пицца (круг) разрезана на `m × n` одинаковых кусочков.

- Есть `m` различных видов начинок (`m ≥ 2`)
- Каждую начинку нужно положить ровно на `n` кусочков (`n ≥ 1`)

**Определение**  
`f(m, n)` = количество способов распределить начинки по кусочкам, если:

- Отражения считаются различными
- Повороты — не считаются различными

Примеры:

- `f(2, 1) = 1`
- `f(2, 2) = 2`
- `f(3, 1) = 2`
- `f(3, 2) = 16`

## Задача

Найти сумму всех `f(m, n) ≤ 10¹⁵`.


## Алгоритм

Количество кусочков `N = m × n`.

По лемме Бёрнсайда:

\[
f(m, n) = \frac{1}{N} \sum_{k=0}^{N-1} \text{Fix}(k)
\]

При повороте на `k` позиций:

- `g = gcd(N, k)`
- Количество циклов = `g`
- Длина цикла = `N/g`

Чтобы раскраска была неподвижна, все позиции в цикле должны иметь одинаковую начинку.

Условия:

- `n % (N/g) == 0`
- `m * (n / (N/g)) == g`

Тогда:

\[
\text{Fix}(k) = \frac{g!}{((n / (N/g))!)^m}
\]

Суммируем по `k`, делим на `N`.

Перебираем все `m ≥ 2`, `n ≥ 1`, пока `f(m, n) ≤ 10¹⁵`.

## Результат

Сумма всех `f(m, n) ≤ 10¹⁵` = **1485776387445623**
